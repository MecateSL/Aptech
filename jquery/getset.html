jQuery methods can be used to either assign or read some value of a selector. A few these methods are text(), html(), attr(), and val(). When these methods are called with no argument it is referred as getters, because it gets (or reads) the values of the element. When these methods are called with a value as an argument it is referred as a setter because it sets (or assign) that value
